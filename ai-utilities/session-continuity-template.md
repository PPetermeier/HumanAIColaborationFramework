## Template Usage Instructions

### For AI Models:
1. **Start each session** by reviewing this template
2. **Update context** as you work with the human
3. **Document outcomes** before ending the session
4. **Prepare handoff** information for future sessions

### For Humans:
1. **Maintain this template** between sessions
2. **Update priorities** as they evolve
3. **Add context** that helps AI understand the project
4. **Review continuity** to ensure nothing important is lost

### Template Maintenance:
- Update after each session
- Archive old sessions but maintain recent context
- Evolve template structure based on what works
- Keep the template focused and practical

*This template is designed to bridge AI memory limitations while supporting the collaborative process outlined in the Human-AI Collaboration Framework.*
